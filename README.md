# ✨ 星迹 StarTrail

**基于 Skyfield 天文计算引擎的 Windows 桌面观星工具**

面向零基础观星小白的全中文桌面应用，调用开放 API 获取实时天文与天气数据。新手模式一键观星，专业模式提供完整天文参数。

## 🚀 快速开始

### 方式一：下载 EXE（推荐，开箱即用）

1. 前往 [Releases](https://github.com/your-username/startrail/releases) 下载最新版 `星迹.exe`
2. 双击运行，无需安装 Python 或任何依赖

### 方式二：从源码运行

```bash
# 1. 克隆或下载项目
cd 星迹

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动
python main.py
```

## 🎮 功能一览

| 功能 | 说明 |
|------|------|
| 🌌 **实时星空** | 根据当前时间/经纬度实时模拟夜空。包含亮星、行星、星座连线。专业模式显示 RA/Dec 网格和恒星名称 |
| 🔭 **观星预报** | 结合天气、云量、月相、湿度，智能评估观星适宜度（优/良/一般/差） |
| 📅 **天文日历** | Google Calendar 风格月视图。有事件的日期以彩色圆形标记。未来一个月可观测事件列表含南北半球标注 |
| ⚙️ **设置** | 新手/专业模式切换、城市定位、API 密钥配置 |

### 新手模式 vs 专业模式

| 新手模式 | 专业模式 |
|---------|---------|
| 简洁卡片界面 | 星空图显示 RA/Dec 网格 |
| 一键观星指数 | 恒星名称 + 赤经赤纬标注 |
| 基本行星列表 | DSO 深空天体标记 |
| 隐藏技术参数 | 事件统计、原始 API 数据 |

### 星空图交互

| 操作 | 功能 |
|------|------|
| 鼠标滚轮 | 缩放 (0.3x ~ 8x) |
| 左键拖拽 | 平移视角 |
| 单击星体 | 弹出信息面板（星等、高度、方位、RA/Dec） |
| 悬停星体 | 高亮光圈 + 参数浮层 |
| 双击 / R 键 | 重置缩放 |

## 🔑 API 密钥配置

应用需要以下免费 API 密钥以获得完整功能：

### 1. OpenWeatherMap（观星预报必需）

- 注册: https://openweathermap.org/api
- 免费套餐: 每分钟 60 次调用
- **注意**: 该 API 在中国大陆可能需要科学上网才能正常访问

### 2. NASA API（每日一图等）

- 注册: https://api.nasa.gov
- 免费套餐: 每小时 30 次（不配置使用 DEMO_KEY 限额更低）

在「设置 → API 密钥」中输入密钥后点击「测试」验证连接。

## 📦 技术栈

| 模块 | 说明 |
|------|------|
| **PySide6** | Qt6 桌面 GUI 框架 |
| **Skyfield** | 精准天文计算（JPL DE421 行星历表） |
| **requests** | HTTP 客户端调用开放 API |
| **PyInstaller** | 打包为独立 EXE |

### 数据来源

| 数据 | 来源 |
|------|------|
| 亮星数据 | Hipparcos 星表（内置 18 颗最亮星） |
| 行星位置 | JPL DE421 历表（通过 Skyfield） |
| 深空天体 | 内置目录（M31, M42, M45 等 10 个） |
| 天气数据 | OpenWeatherMap API |
| 天文图片 | NASA APOD API |
| 自动定位 | ip-api.com |

## 🌍 关于南北半球

天文日历中的事件列表会标注该事件适合哪个半球观测：
- ⬆️ 北半球 — 适合北半球用户
- ⬇️ 南半球 — 适合南半球用户  
- 🌏 全球 — 南北半球均可观测

自动根据设置中的纬度判断你所在半球。

## ⚠️ 已知问题

1. **观星预报需要科学上网** — OpenWeatherMap 在中国大陆的 HTTPS 连接可能超时。应用已自动降级到 HTTP 协议，但部分网络环境仍可能无法连接
2. **Skyfield 历表** — 首次运行时自动下载 JPL DE421 历表（约 17MB）。如果下载失败会使用内置近似算法，行星位置精确度降低但不影响使用
