# 星迹 StarTrail

**基于 Skyfield 天文计算引擎的 Windows 桌面观星工具**

面向零基础观星小白的全中文桌面应用，调用开放 API 获取实时天文与天气数据。新手模式一键观星，专业模式提供完整天文参数。

---

## 快速开始

### 方式一：下载 EXE（推荐）

1. 前往 [Releases](https://github.com/HazardSun/StarTrail/releases) 下载最新版 `StarTrail_vX.X.X.zip`
2. 解压后双击 `StarTrail.exe`，无需安装 Python 或任何依赖
3. 首次启动会自动下载 JPL 星历表（约 17MB），请保持网络通畅

> **系统要求：** Windows 10/11 64 位，建议 8GB+ 内存，支持 OpenGL 3.3+ 的显卡

### 方式二：从源码运行

```bash
# 1. 克隆项目
git clone https://github.com/HazardSun/StarTrail.git
cd StarTrail

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动
python main.py
```

## 功能详解与使用教学

### 1. 实时星空（Sky View）

核心功能，实时模拟当前夜空。

#### 新手模式

简洁卡片界面，上方显示观星指数和今晚推荐目标，下方是实时星空图。

```

│  │  ├─ 木星  -2.7等  高度55°  → 定位    ││
│  │  ├─ 土星  +0.5等  高度32°  → 定位    ││
│  │  └─ M42 星云     高度45°  → 定位    ││
│  └──────────────────────────────────────┘│
│                                          │
│          ★ 星空渲染区域 ★                  │
│    ● 亮星   ● 行星   ─ 星座连线           │
│                                          │
```

#### 专业模式

星空图叠加赤经/赤纬网格，显示恒星名称、深空天体标记。右侧出现四个可折叠面板：

| 面板 | 功能 |
|------|------|
| 🛰️ 卫星追踪 | 实时显示 ISS 等亮卫星位置，近期的过境预报 |
| 🌌 深空天体 | 按星等/高度筛选 M 天体，支持刷新 |
| ✈️ 飞机追踪 | 实时 ADS-B 数据，显示航班号、高度、距离 |
| 🌬️ 大气视宁度 | 当前 seeing 值 + 逐小时预报折线图 |

#### 星空图交互操作

| 操作 | 功能 |
|------|------|
| 🖱️ 鼠标滚轮 | 缩放（0.3x ~ 8x） |
| 🖱️ 左键拖拽 | 平移视角 |
| 🖱️ 单击星体 | 弹出信息面板（星等、高度、方位、RA/Dec） |
| 🖱️ 悬停星体 | 高亮光圈 + 参数浮层 |
| 🖱️ 双击 / R 键 | 重置缩放 |
| 🔘 设备按钮 | 切换裸眼/双筒/望远镜/相机模式，过滤可见星体 |

#### 观测设备模式

四种设备模拟不同观测能力，自动过滤可见天体：

| 设备 | 极限星等 | 适用场景 |
|------|---------|---------|
| 裸眼 | 6.5 | 快速浏览最亮天体 |
| 双筒望远镜 | 10.0 | 寻找彗星、疏散星团 |
| 天文望远镜 | 14.0 | 深空天体观测 |
| 相机 | 18.0 | 天文摄影规划 |

## API 密钥配置

应用需要以下免费 API 密钥以获得完整功能：

### 1. OpenWeatherMap（观星预报必需）

- **注册地址：** https://openweathermap.org/api
- **免费套餐：** 每分钟 60 次调用
- **注意：** 该 API 在中国大陆可能需要科学上网才能正常访问。应用已自动降级到 HTTP 协议，但部分网络环境仍可能无法连接。

### 2. NASA API（每日一天文图）

- **注册地址：** https://api.nasa.gov
- **免费套餐：** 每小时 30 次（不配置则使用 DEMO_KEY，限额更低）

在「设置 → API 密钥」中输入密钥后点击「测试」验证连接。

## 技术栈

| 模块 | 说明 |
|------|------|
| **PySide6** | Qt6 桌面 GUI 框架 |
| **Skyfield** | 精准天文计算（JPL DE421 行星历表） |
| **OpenGL** | 3D 星空渲染（可选 GL 渲染器） |
| **requests** | HTTP 客户端调用开放 API |
| **PyInstaller** | 打包为独立 EXE |

### 数据来源

| 数据 | 来源 |
|------|------|
| 亮星数据 | Hipparcos 星表（内置 18 颗最亮星 + 全部命名恒星） |
| 行星位置 | JPL DE421 历表（通过 Skyfield） |
| 深空天体 | 内置目录（M31, M42, M45, M57, M13 等 20+ 个） |
| 天气数据 | OpenWeatherMap API |
| 天文图片 | NASA APOD API |
| 自动定位 | ip-api.com |
| 飞机追踪 | OpenSky Network / ADS-B Exchange |
| 卫星数据 | Celestrak TLE |

## 常见问题

### Q: 启动后星空图是空白的？
A: 首次启动需要下载 JPL 星历表（约 17MB）。如果下载失败，请检查网络连接，或在设置中手动下载 `de421.bsp` 放入 `data/` 目录。

### Q: 观星指数显示 "--"？
A: 请配置 OpenWeatherMap API 密钥。该 API 需要科学上网才能在中国大陆正常使用。

### Q: 如何更新？
A: 下载最新版 ZIP 解压覆盖即可。配置文件保存在 `~/.startrail/config.json`，升级不会丢失。

### Q: 如何切换回简体中文？
A: 应用界面已为全中文，无需额外设置。

### Q: 专业模式卡顿？
A: 在设置中取消「自动刷新」可降低 CPU 占用。如果显卡支持 OpenGL 3.3+，可开启「GL 渲染器」获得更流畅体验。
